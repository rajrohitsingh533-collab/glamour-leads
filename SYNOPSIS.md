# Project Synopsis

**Project Title:** Glamour Glow - Premium Cosmetic Lead Generation System  
**Category:** Full-Stack Web Development  
**Academic Year:** 2025-2026

## 1. Introduction
In the competitive cosmetic industry, capturing and managing customer interest (leads) is critical for business growth. **Glamour Glow** is a comprehensive solution that combines a high-conversion landing page with a backend CRM (Customer Relationship Management) dashboard.

## 2. Problem Statement
Many small-to-medium cosmetic brands struggle with:
- Managing customer inquiries across multiple platforms (Manual entry).
- Lack of data validation leading to "junk" leads.
- Difficulty in exporting data for marketing campaigns.
- High cost of complex CRM software.

## 3. Proposed System
Our system proposes a centralized web-based platform that:
- Automates lead capture through a responsive, botanical-themed UI.
- Validates data at the entry point using Zod schemas.
- Offers a secure, password-protected Admin Dashboard for real-time monitoring.
- Provides one-click CSV export functionality for easy data portability.

## 4. Key Modules
1. **User Interface Module:** Developed with Next.js and Tailwind CSS for a premium "Apple-like" aesthetic.
2. **Lead Capture Module:** Integrated with Supabase PostgreSQL to ensure persistent and reliable storage.
3. **Authentication Module:** Uses Supabase Auth to protect sensitive business data.
4. **Admin Analytics Module:** A dashboard displaying lead volume, statuses, and search capabilities.

## 5. Technical Stack
- **Framework:** Next.js 15 (App Router)
- **Database:** Supabase (PostgreSQL)
- **Styling:** Tailwind CSS v4
- **Validation:** Zod
- **Infrastructure:** Vercel (Hosting & SSL)

## 6. Future Enhancements
- Integration of a Chatbot for instant customer support.
- Automated email drips for lead nurturing using Resend.
- Advanced analytics charts for skin type/concern distribution.
